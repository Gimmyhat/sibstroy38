# СибСтрой38 — лендинг

Статический сайт. Никакой сборки не нужно — Vercel определит как static site автоматически.

## Файлы

- `index.html` — основной лендинг
- `privacy.html` — Политика обработки персональных данных (152-ФЗ)
- `styles.css` — стили
- `script.js` — интерактив (мобильное меню, FAQ, слайдеры, lightbox альбома АР, загрузка iframe Yandex Forms)
- `img/renders/` — рендеры фасада (1 hero + 3 ракурса)
- `img/plans/` — 5 3D-планировок
- `img/process/` — 20 листов проектной документации (раздел АР)
- `vercel.json` — настройки кеша и заголовков

## Деплой на Vercel — 3 способа

### 1) Drag-drop (самый быстрый)

1. Распакуйте папку `deploy/` на компьютере.
2. Зайдите на https://vercel.com → войдите через GitHub / Google.
3. New Project → Import → перетащите папку.
4. Framework Preset: Other. Deploy.
5. Через ~30 секунд получите ссылку вида `sibstroy38.vercel.app`.

### 2) Через CLI

```bash
cd deploy
npx vercel
```

### 3) Через GitHub

1. Создайте репозиторий, залейте содержимое `deploy/`.
2. На vercel.com → New Project → Import Git Repository → выберите репо.
3. Deploy.

## Кастомный домен

Vercel → Project → Settings → Domains → Add. Далее в DNS вашего регистратора (reg.ru / nic.ru / cloudflare) добавьте запись `A → 76.76.21.21` или `CNAME → cname.vercel-dns.com`.

## ⚠️ ВАЖНО: 152-ФЗ и приём заявок

С 01.07.2025 первичный сбор персональных данных граждан РФ должен происходить **на серверах в России**. Поэтому форма заявки на сайте **не должна слать данные в Formspree, Web3Forms, Google Forms или любые иностранные сервисы** — это нарушение 242-ФЗ и риск штрафа до 700 000 ₽.

**Используемое решение:** Yandex Forms (сервера в РФ).

### Как подключить Yandex Forms — 5 минут

1. Зайдите на https://forms.yandex.ru/ под учётной записью Яндекс (или создайте новую — на корпоративный email).
2. **Создать форму** → пустая форма. Добавьте поля:
   - Текст: «Ваше имя» (обязательно)
   - Телефон: «Телефон» (обязательно, с маской `+7 ___ ___-__-__`)
   - Текст: «Адрес участка или кадастровый номер» (опционально)
   - Число: «Желаемая площадь, м²» (опционально)
   - Длинный текст: «Коротко о доме» (опционально)
   - **Чекбокс согласия:** «Я даю согласие на обработку персональных данных в соответствии с [Политикой обработки ПДн](https://ваш-домен/privacy.html)» (обязательно, **без предустановленной галочки** — это требование с 01.09.2025)
3. Настройки формы:
   - **Уведомления → Email** → впишите `sibstroy-sms@yandex.ru`. Все заявки будут приходить туда.
   - **Внешний вид** → выберите тёмную тему (она лучше сочетается с дизайном секции CTA на сайте) или оставьте дефолтную.
   - **Доступ** → «Открытая форма»
4. **Опубликовать** → скопируйте ссылку для встраивания (вид: `https://forms.yandex.ru/cloud/ABCD1234.../?iframe=1`)
5. Откройте `deploy/index.html`, найдите строку:
   ```html
   data-yandex-form-src="https://forms.yandex.ru/cloud/REPLACE_WITH_YOUR_FORM_ID/?iframe=1"
   ```
   и замените URL на ваш реальный из п. 4. После этого скрипт автоматически загрузит форму вместо плейсхолдера.
6. Redeploy на Vercel.

### Альтернативные приёмники (если Yandex Forms не подойдёт)

| Сервис | Сервера | Бесплатно | Когда выбрать |
|---|---|---|---|
| Bitrix24 веб-формы | РФ | ✓ | Если нужна полноценная CRM, а не только email |
| amoCRM RU | РФ | базовый тариф | Тоже CRM-вариант |
| getCourse | РФ | по тарифу | Если планируете онлайн-курсы/лиды |
| Tilda Forms | РФ | в составе Tilda | Если параллельно используете Tilda |
| Собственный VPS в РФ | РФ | от 200 ₽/мес | Selectel/Timeweb/Beget + минимальный PHP/Node-эндпоинт |

**❌ НЕЛЬЗЯ:** Formspree, Web3Forms, Google Forms, Typeform, Mailchimp Forms — все на иностранных серверах.

## ⚠️ Уведомление Роскомнадзора

До запуска сайта в проде ООО «Защита» обязано подать уведомление в РКН о начале обработки персональных данных (ст. 22 152-ФЗ). Штраф за неподачу — до 300 000 ₽ для юридического лица.

Подать можно онлайн через личный кабинет на https://rkn.gov.ru/ (раздел «Уведомления») или через Госуслуги. Срок рассмотрения — ~30 дней.

## Что доделать перед продом

1. **Yandex Forms** — см. инструкцию выше. Без этого форма не работает.
2. **Уведомление в РКН** — см. выше.
3. **Документы.** В подвале раздел «Документы» ведёт на mailto (запрос по email). Альтернатива — загрузить настоящие PDF в `deploy/docs/` и заменить ссылки:
   - `docs/sro.pdf` — выписка из СРО
   - `docs/license-mchs.pdf` — лицензия МЧС
   - `docs/contract-template.pdf` — образец договора
4. **Метрики (опционально).** Если будете подключать Яндекс.Метрику — добавьте код перед `</body>` и обновите раздел «10. Cookie» в `privacy.html`. Дополнительно нужен cookie-баннер с возможностью отказа.
5. **Реальные реквизиты ООО «Защита»** в `privacy.html` — раздел «1. Оператор персональных данных» сейчас содержит шаблон, нужно вписать ИНН и ОГРН.

## Производительность

Все шрифты загружаются с Google Fonts CDN. CSS и JS — отдельные файлы для кеширования (год). HTML — без кеша, чтобы правки прилетали сразу. Изображения lazy-loaded.
